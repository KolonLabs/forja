# Registros de beat — Relato

La escena declara el arco tonal. Un beat solo lleva `[registro: ...]` cuando necesita desviarse o intensificarlo.

| Registro | Uso |
|---|---|
| `contenido` | Normalidad, observación o preparación. |
| `tenso` | Anticipación, amenaza o fricción. |
| `revelación` | Información que altera la lectura de la situación. |
| `íntimo` | Cercanía emocional o corporal sin énfasis gráfico. |
| `explícito / visceral` | Sexualidad o corporalidad gráfica en primer plano. |
| `brutal` | Fuerza, impacto o violencia sin amortiguación. |
| `elipsis` | Salto deliberado que evita desarrollar un intervalo. |

El registro no obliga a un número de palabras ni a una fórmula expresiva. Si no aparece, el escritor sigue el arco tonal de la escena.
